使用说明

1. 获取GitHub Token

  • • 访问 GitHub Settings
  • • 点击 "Generate new token"
  • • 选择 "user:follow" 权限
  • • 复制生成的token

2. 输入用户名

  • • 每行输入一个GitHub用户名
  • • 支持批量输入，最多100个
  • • 系统会自动去重
  • • 建议设置适当的请求间隔
